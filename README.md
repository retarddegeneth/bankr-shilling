# bankr-shilling
> have you shilled today?

Terminal-themed Bankr $BNKR shill tracker. Top 10 shillers get paid daily.

## local dev
```bash
cd web && python3 app.py
```

## github pages
Open the live page under repository Settings > Pages.

## deploy
- manual payout in /payments
- leaderboard in /
